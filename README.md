# SafeDR-Agent

SafeDR-Agent is a clinician-in-the-loop diabetic retinopathy screening framework.

The project focuses on:

- retinal fundus image classification
- uncertainty estimation
- image-quality-aware decision making
- abstention policy
- referral recommendation
- Grad-CAM explainability
- safety-aware evaluation

## Main Dataset

Kaggle dataset:

`sovitrath/diabetic-retinopathy-224x224-2019-data`

## Repository Status

This repository is synchronized from Kaggle for reproducible experimentation.
